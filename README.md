## ✨ Overview

Drona Hub is a centralized platform designed to help engineering students access important academic and career resources from one place.

The platform provides notes, syllabus, PYQs, career roadmaps, coding resources, CGPA calculator, AI chatbot, and interactive features through a modern and responsive interface.

---

## 🚀 Features

* 📚 **Notes, Syllabus & PYQs**
  Semester-wise academic resources with direct PDF and Drive links

* 🛣️ **Career Roadmaps**
  Detailed roadmaps for AI/ML, Web Development, Cloud, Cybersecurity, Data Science, and more

* 💻 **Coding Resources**
  Curated coding channels, websites, and learning resources for beginners

* 📊 **CGPA Calculator**
  Dynamic semester-wise GPA/CGPA calculator

* 🤖 **AI Chatbot**
  Gemini-powered assistant for student-related queries

* 🌙 **Dark / Light Mode**
  Smooth theme switching with saved preferences

* 📱 **Responsive Design**
  Optimized for desktop, tablet, and mobile devices

* ✨ **Modern UI**
  Glassmorphism design with animations and interactive effects

* 🎮 **Mini Games**
  Includes Tic Tac Toe and Sudoku for quick refreshment

* 🔍 **Search & Filters**
  Easy navigation across resources and roadmaps
